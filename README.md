# Alumni Web Portal

A Java fullstack web application to build a bridge between Alumni and the current student body

## Technologies used:
- Java
- J2EE
- JSPs and Servlets
- JDBC
- MySQL
- HTML
- Bootstrap

## Features
- Overview page
- Alumni register
- update details
- Authentication
- Authorization
- various dynamic filtered view pages
- CRUD operations
